# MSEAVIEW

Upload and view multiple sequence alignments, phylogenetic trees and more in your browser.

## Setup

### Frontend (React + Vite + Tailwind)
```bash
cd frontend
npm install
npm run dev
```

Visit `http://localhost:5173` and upload your files.
